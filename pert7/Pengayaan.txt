Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa perbedaan antara interface dengan class biasa?
2. Mengapa interface tidak memiliki implementasi method di dalamnya?
3. Apa tujuan penggunaan keyword "implements" pada sebuah class?
4. Apakah sebuah class dapat mengimplementasikan lebih dari satu interface? Jelaskan dan berikan contoh sederhana.
5. Jika sebuah class mengimplementasikan interface, apa yang terjadi jika class tersebut tidak mengimplementasikan semua method dari interface tersebut?

Jawaban:
1. Interface hanya berisi kontrak (deklarasi method) tanpa implementasi, sedangkan class biasa memiliki atribut, method lengkap, dan bisa langsung dibuat objeknya.
2. Interface tidak memiliki implementasi method karena tujuannya adalah mendefinisikan aturan atau kontrak yang harus dipenuhi class yang mengimplementasikannya.
3. Keyword implements digunakan untuk menyatakan bahwa sebuah class setuju memenuhi seluruh kontrak yang ada di dalam interface tersebut, yaitu dengan menyediakan implementasi methodnya.
4. sebuah class dapat mengimplementasikan lebih dari satu interface dan memungkinkan class memiliki beberapa perilaku wajib. ex: class Robot implements Gerak, Suara { }
5. class tersebut otomatis menjadi abstract dan tidak dapat dibuat objeknya sampai semua method interface diimplementasikan.
