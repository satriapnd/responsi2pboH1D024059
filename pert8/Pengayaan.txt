Setelah anda menyelesaikan kode, tolong jawab pertanyaan dibawah ini ya =


1. Apa tujuan utama adanya interface dalam pemrograman berorientasi objek, dan bagaimana interface membantu menciptakan kontrak perilaku?
2. Mengapa interface dapat memiliki banyak implementasi, dan apa makna konseptual dari kemampuan tersebut dalam desain sistem?
3. Apa perbedaan mendasar antara extends dan implements dalam konteks interface, jika dilihat secara konsep tanpa melihat contoh kode?
4. Apa alasan diperkenalkannya default method dalam interface, dan bagaimana fitur ini memengaruhi keberlangsungan (compatibility) suatu sistem yang sudah berjalan?
5. Bagaimana konsep multiple interface memperluas fleksibilitas desain tanpa menimbulkan masalah kompleksitas yang biasa muncul pada multiple inheritance?

Jawaban:
1. Interface bertujuan mendefinisikan kontrak perilaku yang wajib dipatuhi class yang mengimplementasikannya. Dengan interface, sistem memiliki standar perilaku yang konsisten tanpa memaksa struktur data atau implementasi tertentu.
2. Interface dapat memiliki banyak implementasi karena hanya mendefinisikan perilaku, bukan cara kerjanya. Secara konsep, ini memungkinkan berbagai class berbeda menjalankan fungsi yang sama dengan cara mereka sendiri, sehingga desain menjadi lebih fleksibel dan mudah diperluas.
3. extends berarti mewarisi sifat atau struktur dari induk (konsep pewarisan), sedangkan implements berarti memenuhi kontrak perilaku yang ditentukan interface (konsep kewajiban). Extends mewarisi, implements mematuhi.
4. Default method diperkenalkan agar interface dapat menambah perilaku baru tanpa memaksa semua class lama mengubah kode mereka. Hal ini menjaga kompatibilitas sistem sehingga perubahan pada interface tidak merusak implementasi yang sudah berjalan.
5. Multiple interface memberi fleksibilitas karena class bisa memenuhi beberapa kontrak sekaligus tanpa benturan seperti pada multiple inheritance (misalnya konflik atribut). Interface tidak membawa keadaan (state), sehingga tidak menimbulkan kompleksitas pewarisan ganda, tetapi tetap memberikan banyak perilaku yang bisa digabung.
