Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa fungsi keyword 'extends' dalam deklarasi class Java?
2. Jelaskan perbedaan antara Method Overloading (yang ada di latihan sebelumnya) dengan Method Overriding (yang ada di latihan ini)!
3. Apa kegunaan keyword 'super' di dalam constructor subclass (Manajer)?
4. Jika variabel di class Karyawan diubah access modifier-nya menjadi 'private', apa yang akan terjadi pada class Manajer? Bagaimana solusinya?
5. Mengapa kita perlu menggunakan Inheritance? Apa keuntungan utamanya dibandingkan membuat dua class yang terpisah sama sekali (misal class Karyawan dan class Manajer dibuat terpisah tanpa hubungan)?

Jawaban:
1. Keyword extends digunakan untuk menyatakan bahwa sebuah class adalah turunan (subclass) dari class lain. Ini membuat subclass mewarisi semua atribut dan method dari superclass sehingga dapat digunakan atau dioverride.
2. Method Overloading adalah membuat beberapa method dengan nama sama tetapi parameter berbeda dalam satu class. Sementara Method Overriding adalah mengganti ulang isi method dari superclass di subclass dengan implementasi yang berbeda, tetapi nama dan parameternya sama.
3. Memanggil constructor milik superclass. Ini memastikan atribut yang dimiliki class induk bisa diinisialisasi lebih dulu sebelum atribut khusus subclass.
4. Jika variabel di class Karyawan diubah menjadi private, class Manajer tidak bisa mengaksesnya secara langsung. Solusinya adalah menyediakan getter dan setter di class Karyawan agar subclass tetap bisa mendapat atau mengubah nilai variabel tersebut secara aman.
5. Inheritance diperlukan untuk menghindari duplikasi kode dan membuat struktur program lebih rapi. Dengan mewarisi class Karyawan, class Manajer dapat menggunakan kembali atribut dan method yang sama tanpa menulis ulang, sehingga kode lebih efisien, terorganisasi, dan mudah diperluas.
